ESPHome component to monitor a Atorch DL24 dummy load via bluetooth

## Supported devices

* Atorch DL24
* Atorch DL24P

## Untested devices using the same protocol

* Atorch AT24
* Atorch AT3010
* Atorch DT24
* Atorch DT3010
* Atorch UD18
* Atorch UD18-L
* Atorch T18
* Atorch J7-C
* Atorch J7-H

## Unsupported devices

* Atorch Q7-T
* Atorch A3-B
* Atorch KM001



This esphome component is work in progress!
